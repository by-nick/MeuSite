Jogo Da Cobrinha

Descrição

Utilizando a linguagem Python, recriei o famoso Jogo da Cobrinha.

Controles

O jogo permite o player controlar uma cobra que se move pela tela utilizando W/A/S/D, devendo coletar maçãs para crescer enquanto evita colidir com o próprio corpo. A cada alimento consumido, a pontuação aumenta.

Biblioteca e Editor

Para desenvolver, utilizei a biblioteca Pygame, que facilitam a criação da interface visual e o controle de animações e o editor de código VSCode.
